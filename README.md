## NameCheck GH
Search Ghana's official business name registry instantly.

### Tech Stack
- React + Vite (frontend)
- Vercel Serverless Functions (backend)
- Axios + Cheerio (ORC scraping)
- Tailwind CSS (styling)
- Framer Motion (animations)

### Run Locally
1. Clone the repo
2. `npm install`
3. `npm run dev` (frontend)
4. `vercel dev` (to run serverless functions locally)

### Deploy
- Push to GitHub
- Connect repo to Vercel
- Framework: Vite
- Build command: `vite build`
- Output directory: `dist`
- No environment variables needed
